# spotifymeta

A local explorer for the `spotify_2025_07_metadata` archive.

Reads `spotify_clean.sqlite3` and `spotify_clean_audio_features.sqlite3` from the current working directory.

## Dev

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.tsx
```

This project was created using `bun init` in bun v1.3.4. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
